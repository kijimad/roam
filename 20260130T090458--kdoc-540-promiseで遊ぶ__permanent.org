* 概要

[[https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Global_Objects/Promise][Promise - JavaScript | MDN]]が秀逸だったのでメモする。

#+caption: Promise でresolveが実行される
#+begin_src js :results output
const promise = new Promise((resolve, reject) => {
  resolve(123);
})

promise.then((res) => {
  console.log(res)
})
promise.catch((err) => {
  console.log("これは呼び出されない")
})
#+end_src

#+end_src
