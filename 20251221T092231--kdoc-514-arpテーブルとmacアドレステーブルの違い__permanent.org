* 概要

ARPテーブルとMACアドレステーブルはどちらもMACアドレスを使う。違いを調べることにした。

- ARPテーブル :: MAC と IP ... IPアドレスを引く用。各NICデバイスごとで持つ
- MACアドレステーブル :: MAC と 物理ポート ... ハードウェアの物理ポートを引く用。スイッチングのために使う
