概要

任天堂のJoy-ConをPC(Ubuntu)の読書デバイスとして使っている。片手・ワイヤレスでページ送りできて便利である。

手順

Ubuntuで設定する手順をメモした。初期状態では、Bluetooth接続しても一瞬で切断された。

まずDanielOgorchock/joycondをインストールする。手順はリポジトリに書いてあるとおり。

つぎに、デフォルトのドライバを無効化する。 

sudo modprobe -r hid_nintendo

再起動すると、Bluetooth接続できた。しかしantimicroxで認識しなかった。 sudo antimicrox するか、inputグループを追加する。 

sudo usermod -aG input $USER
