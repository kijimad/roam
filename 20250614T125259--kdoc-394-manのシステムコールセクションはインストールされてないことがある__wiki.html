概要

Linux Mint 22.1で、manで検索したときにシステムコールのセクションが見つからないことに気づいた。ディストリビュージョンによっては、省略されていることがあるようである。

インストールする。 

sudo apt update
sudo apt install manpages-dev

man 2 fork | head -n10

fork(2)                       System Calls Manual                      fork(2)

NAME
       fork - create a child process

LIBRARY
       Standard C library (libc, -lc)

SYNOPSIS
       #include <unistd.h>

なし。
