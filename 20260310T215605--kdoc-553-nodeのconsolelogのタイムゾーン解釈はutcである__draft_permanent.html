概要

Nodeで動かすJavaScript上で、Dateのタイムゾーンが時刻フォーマットによって変わる奇妙な仕様を見た。確認するため console.log を使ったところ、出力がUTCなのに気づいた。ブラウザでは console.log はローカル時間で表示されるので、Nodeの仕様に見える。 

echo $TZ








console.log(new Date('2026-06-01')) // UTC -> UTC
console.log(new Date('2026-06-01T00:00:00')) // local timezone(Asia/Tokyo) -> -9 -> UTC








2026-06-01T00:00:00.000Z
2026-05-31T15:00:00.000Z
undefined







より端的に示す。






console.log(new Date(2026,0,1))








2025-12-31T15:00:00.000Z
undefined







たくさんの落とし穴がありそうである。
